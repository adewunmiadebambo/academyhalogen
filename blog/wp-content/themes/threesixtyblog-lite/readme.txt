ThreeSixtyBlog Lite is a modern & functional blog WordPress theme.

== Description ==
ThreeSixtyBlog Lite is clean, clutter free design is the perfect choice for the minimal loving creative blogger. The blog theme offers a stunning and awesome visual experience to showcase your content with responsive grid system layout optimized for mobile touch and swipe. A beautiful responsive theme for bloggers with variety options to change Logo(Text or Image), Background color, Brand color, Link color and Main color from nine pre-difined colors, headers image, Header media(Video or image) and sidebars easily.


== Installation ==
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in ThreeSixtyBlog in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
